KEY INFORMATION:

COMMAND 3: exec 3<app.log
Opened app.log for reading and assigned it to file descriptor 3. This doesn't produce output but establishes the file descriptor.

COMMAND 4: lsof -p $$ | grep app.log
Lists only the current bash process's open files:
lr-x------ 1 codespace codespace 64 Jul  3 14:00 3 -> /workspaces/SGA-1/q4/app.log

Shows that file descriptor 3 is now open for reading (lr-x--) the app.log file.

COMMAND 6: ls -l > stdout.txt
Redirected standard output to stdout.txt using > operator. The directory listing is written to the file instead of displayed on terminal.

COMMAND 7: ls /not_a_real_path 2> stderr.txt
Attempted to list a non-existent directory and redirected the error message (stderr) to stderr.txt using 2> operator.

COMMAND 8: (ls -l && ls /not_a_real_path) > combined.txt 2>&1
Combined both stdout and stderr into combined.txt using > and 2>&1 operators. The successful ls output and the error message are both captured in one file.
